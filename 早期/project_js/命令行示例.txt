bin/kafka-console-producer.sh --broker-list PLAINTEX/10.143.62.232:9092 --topic temp

bin/kafka-console-consumer.sh --bootstrap-server PLAINTEXT://10.143.62.232:9092 --topic temp

从Windows上传到Linux
rz 

从Linux下载到Windows
sz filename

docker ps 查看正在运行的容器

docker ps -a 全部容器

docker images 列出所有镜像

docker start CONTAINER_ID 启动容器

docker exec -it b58b0f8d902f /bin/bash 进入容器

查看容器绑定和映射的端口及IP地址： docker port b58b0f8d902f

获取容器的PID为96090： docker inspect -f "{{.State.Pid}}" b58b0f8d902f

docker cp /opt/tensorboard_trail b58b0f8d902f:/wzs 将文件导入容器中

docker container cp -a  b58b0f8d902f:/wzs/tensorboard_trail/model.log /opt/tensorboard_trail/ 将容器中的文件复制到Linux主目录下


python -c "import kafka;print(kafka)" 查看python包安装位置

tar xzvf kafka-1.3.5.tar.gz -C kafka_package/   解压到指定文件
