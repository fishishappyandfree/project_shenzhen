技术：
	如果需要把Python项目所需要的库生成一个文件，方便下次使用时用命令直接安装，可以使用pip freeze
	pip freeze >> requirements.txt
	生成一个迁移文件
	然后通过命令：
	pip install -r requirement_path
	即可重新安装环境
